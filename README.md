This repository contains code for experimental materials pertaining to a study of logical primitives in human thought (joint work with [Emmanuel Chemla](http://www.emmanuel.chemla.free.fr/) and [Manuel Kri≈æ](http://manuel.kriz.fr/)).

The experiment uses the [Ibex](http://spellout.net/ibexfarm/) platform for running psycholinguistic experiments. See the [Ibex repository](https://github.com/addrummond/ibex) for license details, documentation, etc.
